### 最新版本说明 3.0.0（开发中预计6月底发布）

+ API
  + Swagger2
  + example.org/oc3/swagger-ui.html

+ 重要数据加密支持
  + 配置文件加密（密码, AK）
  + 数据库密码字段加密

+ 自动化运维特性
  + 外部平台或工具原生API调用
  + 多版本API支持（逐步升级）
  + 以用户纬度数据映射全自动配置管理
  
+ 核心功能
  + 堡垒机(Jumpserver全自动运维)
  + 可视化数据管理DMS（支持AliyunRDS, 自建Mysql）
    + 权限申请
    + 操作审计
    + 高危操作过滤
    + 自动导出
    + 字段脱敏
  + Zabbix(API4.0)全自动运维
  + 多云支持
    + 阿里云
    + AWS
    + 腾讯云
    + 私有云Vmware-vSphere
