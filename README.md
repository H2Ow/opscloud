### 最新版本说明 3.0.0

+ API
  + Swagger2
  + http://example.org/oc3/swagger-ui.html

+ 重要数据加密支持
  + 配置文件加密（密码，Token, AK）
  + 数据库密码字段加密
  + 即使被拖库或源码泄漏也不会丢失重要信息

+ 自动化运维特性
  + 外部平台或工具原生API调用
  + 多版本API支持（逐步升级）
  + 以用户纬度数据映射全自动配置管理
  
+ 操作审计
  + 堡垒机
  + DMS（有计划开发RDS操作审计）
